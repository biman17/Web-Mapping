ABOUT OUR WEBSITE

The photogrammetric products developed on this website are a Digital Surface Model (DSM) and Digital Terrain Model (DTM) using the process of aerial image matching for the area Boditi,Ethiopia. These products are available in the form of raster representation. Contour lines for each of them are added as vector layers.

These products are essential for applications such as,

  • Urban cadastre
  • Flood modelling
  • 3D city modelling
  • Crisis management

What is a DSM?

An elevation model that includes the tops of buildings, trees, powerlines, and any other objects. Commonly this is seen as a canopy model and only 'sees' ground where there is nothing else covering it.

View DSM Map


What is a DTM?

DTM can be described as a 3D representation of a terrain surface consisting of X, Y, Z coordinates stored in digital form. It includes not only heights and elevations but other geographical elements and natural features (such as rivers, mountains etc.).

View DTM Map

Geo-Web Products

The maps below show DTM and DSM for the city of Boditi, Ethiopia rendered using geo-web services. Both the products are of 5m resolution. It also includes contour lines drawn at an interval of 15m for both the rasters. Hover the mouse on displayed rasters to see the feature information in the "Feature Info" tab. You can also enable/disable raster layers and switch between the OSM and Bing baselayers.

3D Visualization

The above maps are further visualized as 3D models of DTM and DSM. Since the study area is a flat land, the variations in the elevation across the layer is not very prominently visible. Hence elevation data was interpolated accordingly to show the variations in the terrain. Qgis2threejs plugin on QGIS software was used to render the 3D views which uses WebGL(Web Graphics Library), a Javascript API for rendering high performance interactive 3D and 2D graphics with any comaptible web browser.
